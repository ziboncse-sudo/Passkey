# 🔐 পাসওয়ার্ড ভল্ট — Flutter Password Manager

একটি সম্পূর্ণ অফলাইন, এনক্রিপ্টেড পাসওয়ার্ড ম্যানেজার অ্যাপ।

---

## ✨ ফিচার সমূহ
- 🔒 **PIN লক** — অ্যাপ খুলতে ৪-সংখ্যার পিন
- 💾 **লোকাল স্টোরেজ** — সব ডেটা শুধু ডিভাইসে থাকে
- 🛡️ **AES-256 এনক্রিপশন** — পাসওয়ার্ড সম্পূর্ণ নিরাপদ
- 🗂️ **ক্যাটাগরি** — সোশ্যাল, ইমেইল, ব্যাংকিং, কাজ, শপিং, অন্যান্য
- 🔑 **পাসওয়ার্ড জেনারেটর** — শক্তিশালী পাসওয়ার্ড তৈরি করুন
- 📊 **পাসওয়ার্ড স্ট্রেংথ মিটার** — পাসওয়ার্ড কতটা শক্তিশালী দেখুন
- 📋 **কপি বাটন** — এক ক্লিকে কপি
- 🔍 **সার্চ** — যেকোনো পাসওয়ার্ড খুঁজুন
- ↔️ **Swipe অ্যাকশন** — বাম দিকে সোয়াইপ করে সম্পাদনা/মুছুন

---

## 🚀 ইনস্টলেশন

### প্রয়োজনীয়তা
- Flutter SDK 3.0+
- Android Studio / VS Code
- Android device বা Emulator (API 21+)

### ধাপসমূহ

```bash
# ১. প্রজেক্ট ডিরেক্টরিতে যান
cd password_manager

# ২. Dependencies ইনস্টল করুন
flutter pub get

# ৩. অ্যাপ রান করুন
flutter run

# ৪. APK বিল্ড করুন (রিলিজ)
flutter build apk --release
```

## 📁 ফাইল স্ট্রাকচার

```
lib/
├── main.dart                    # অ্যাপ এন্ট্রি পয়েন্ট
├── models/
│   └── password_entry.dart      # ডেটা মডেল
├── screens/
│   ├── lock_screen.dart         # PIN লক স্ক্রিন
│   ├── home_screen.dart         # হোম স্ক্রিন

├── widgets/
│   ├── password_card.dart       # পাসওয়ার্ড কার্ড
│   ├── category_chip.dart       # ক্যাটাগরি চিপ
│   └── detail_sheet.dart        # বিস্তারিত শিট
├── services/
│   ├── database_service.dart    # SQLite ডেটাবেজ
│   ├── encryption_service.dart  # AES এনক্রিপশন
│   └── password_provider.dart   # স্টেট ম্যানেজমেন্ট
└── utils/
    └── app_theme.dart           # থিম ও রং
```

## 🔧 ব্যবহৃত প্যাকেজ

| প্যাকেজ | কাজ |
|---------|-----|
| `sqflite` | লোকাল ডেটাবেজ |
| `encrypt` | AES-256 এনক্রিপশন |
| `flutter_secure_storage` | নিরাপদ কী স্টোরেজ |
| `provider` | স্টেট ম্যানেজমেন্ট |
| `google_fonts` | বাংলা ফন্ট |
| `flutter_slidable` | Swipe অ্যাকশন |
| `shared_preferences` | PIN সংরক্ষণ |

---

## 🛡️ নিরাপত্তা

- পাসওয়ার্ড **AES-256-CBC** দিয়ে এনক্রিপ্ট করা হয়
- এনক্রিপশন কী **Android Keystore**-এ সংরক্ষিত
- অ্যাপ **অফলাইন** — কোনো ইন্টারনেট সংযোগ নেই
- ডেটা ব্যাকআপ **বন্ধ** (`android:allowBackup="false"`)
